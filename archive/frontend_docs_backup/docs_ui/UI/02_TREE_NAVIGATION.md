# Tree Navigation System

## Purpose

To represent the full Vedic corpus as a hierarchical structure.

---

## Data Source

* vedic_corpus_registry.json

---

## Features

* Recursive tree rendering
* Expand / collapse nodes
* Lazy loading for deep levels
* Highlight active node

---

## Interaction

* Click node → load content in center
* Expand arrow → reveal children

---

## Example Structure

Sruti → Rig Veda → Samhita → Mandala → Sukta → Verse
