# Layout System

## Global Layout

Three-column structure:

* Left: Knowledge Tree
* Center: Content Viewer
* Right: Application Panel

---

## Layout Grid

* 12 column grid:

  * Left: 3 columns
  * Center: 6 columns
  * Right: 3 columns

---

## Top Bar

Includes:

* Logo
* Navigation (Library, Education, Guidance, Community)
* Search
* Language switcher
* User login

---

## Responsiveness

* Desktop: Full 3-column layout
* Tablet: Collapsible right panel
* Mobile: Tree → Drawer
