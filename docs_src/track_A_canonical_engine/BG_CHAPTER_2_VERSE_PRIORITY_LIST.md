# BG_CHAPTER_2_VERSE_PRIORITY_LIST.md
Bhagavad-Gītā Chapter 2 Verse Priority List

Track A — Dataset Pilot Construction

---

# 1. Purpose

This document defines the **priority order for constructing the Chapter 2 dataset**.

Rather than entering all 72 verses immediately, the system begins with the **most philosophically foundational verses**.

These verses establish the **core conceptual architecture of the Bhagavad-gītā**.

---

# 2. Priority Philosophy

The priority verses were selected based on:

• philosophical significance  
• frequency of citation in commentaries  
• conceptual richness  
• influence on later chapters  
• importance in Vedānta discussions  

---

# 3. Priority Tier System

Verses are divided into three tiers:

Tier 1 — Core Philosophical Verses  
Tier 2 — Major Supporting Verses  
Tier 3 — Contextual Verses

# 10. Data Entry Strategy

For each priority verse, create:

verse JSON file concept tags commentary entries cross references verse analysis
This allows the pilot system to demonstrate the **full knowledge architecture**.

# 14. Final Principle

The goal is not merely to digitize the Bhagavad-gītā.

The goal is to **structure its philosophical knowledge so it can be explored, studied, and connected across the entire Vedic corpus**.
