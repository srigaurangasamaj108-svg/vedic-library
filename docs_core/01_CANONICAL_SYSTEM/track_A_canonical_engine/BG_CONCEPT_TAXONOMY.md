# Bhagavad-Gītā Concept Taxonomy
Track A — Conceptual Layer

---

# 1. Purpose

The Bhagavad-Gītā Concept Taxonomy defines the philosophical vocabulary
that appears throughout the Gītā.

This taxonomy enables:

• concept tagging  
• knowledge graph creation  
• curriculum design  
• philosophical exploration  
• guidance systems  

Instead of treating verses as isolated text, the taxonomy reveals
the **conceptual architecture of the Gītā**.

---

# 2. Concept Hierarchy

Concepts are organized hierarchically.


Metaphysics
Ethics
Psychology
Yoga
Devotion
Knowledge
Cosmology
Liberation


Each concept belongs to a category.
