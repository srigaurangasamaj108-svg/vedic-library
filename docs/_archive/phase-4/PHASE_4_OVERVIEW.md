### Why Search Is Deferred Beyond Phase 4

Phase 4 focuses on:
- Context
- Structure
- Guided understanding
- Educational framing

Search, by contrast, introduces:
- Non-linear access
- Attention redirection
- Discovery power

Introducing search **before** educational context risks:
- Fragmentation of understanding
- Cherry-picking of verses
- Implicit prioritization of content

Therefore, search is **intentionally deferred**.

---

### How Phase 4 Prepares for Search (Without Implementing It)

Phase 4 prepares search responsibly by:

- Defining themes and concepts
- Establishing educational pathways
- Clarifying boundaries between:
  - Canonical text
  - Commentary
  - Educational guidance
- Creating metadata that can later support scoped discovery

No search engine, index, or UI is introduced in Phase 4 itself.

---

### Constitutional Rule

No search functionality may be implemented unless:

- Phase 4 is complete
- Phase 4 educational structures exist
- Phase 4.5 is explicitly opened
- `SEARCH_AND_DISCOVERY_OVERVIEW.md` is fully respected

This rule protects contemplative reading and scholarly integrity.

---

> **Education must precede discovery.  
> Context must precede search.**
