# Vedic Library — Bhagavad-gītā Reader

## Version
**v1.0-pilot**

---

## What This Is

This is the **Phase-1 pilot interface** of the Vedic Library project.

It provides a **calm, faithful, read-only reading experience**
of the **Bhagavad-gītā**, strictly governed by the architectural
and ethical foundations established in Phase-0.

---

## Included in v1.0-pilot

- Bhagavad-gītā only
- Verse-by-verse navigation
- Canonical Sanskrit (Devanāgarī)
- IAST transliteration
- Word-for-word synonyms
- English translation
- English purport (Śrīla Prabhupāda)
- Editorial-unit–governed grouping
- No required login
- No tracking
- No personalization
- No algorithmic authority

---

## Explicitly Excluded

The following are **not present** in v1.0-pilot:

- Search
- User accounts
- Donations
- AI features
- Multiple ācāryas
- Progress tracking
- Learning paths
- Recommendations
- Analytics

Any such features belong to **later phases only**.

---

## Architectural Discipline

- All content is JSON-driven
- Canonical text is immutable
- Editorial Units define semantic scope
- Verse Index defines availability
- UI never invents meaning
- Absence is represented by silence

---

## Purpose of This Version

The purpose of `v1.0-pilot` is to answer one question only:

> **Can a human sit with the Bhagavad-gītā  
> and read it peacefully, without distortion or pressure?**

If the answer is “yes”, Phase-1 is successful.

---

## Status

**Phase-1 complete.**  
This version is **feature-frozen**.

Further expansion will occur only in later phases
according to the project constitution.
