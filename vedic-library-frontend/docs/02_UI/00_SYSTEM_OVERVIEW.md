# Vedic Knowledge System – Frontend Overview

## Purpose

This system is designed as a **Vedic Knowledge Interface**, integrating:

* Canonical Śāstra (Library)
* Education (Courses)
* Guidance (Life application)
* Community (Seva domains)

---

## Core Principle

**Verse is the atomic unit of knowledge.**

All features connect to:

* Verses
* Text units
* Knowledge nodes

---

## System Axes

### 1. Canonical Axis (Tree)

* Structured hierarchy of Vedic corpus

### 2. Content Axis (Center)

* Verse display
* Commentary system

### 3. Application Axis (Right Panel)

* Courses
* Guidance
* Seva domains

### 4. Filter Axis (Top)

* Age groups
* Categories
* Language

---

## UI Philosophy

* Minimal
* Text-first
* Deep navigation
* Context-rich
